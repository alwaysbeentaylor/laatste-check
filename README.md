# Website Intake Form - Tanoshi Sushi

Dit is een interactieve, mobile-first intake tool gebouwd voor Tanoshi Sushi om de laatste details voor hun nieuwe website te verzamelen.

## Features
- **Premium Design:** Glassmorphism, animaties en een luxe uitstraling.
- **Wizard Flow:** Stap-voor-stap proces voor betere focus.
- **Telegram Integratie:** Stuurt de antwoorden direct naar een Telegram chat via een bot.
- **No-Config Deploy:** Klaar om als statische site op Vercel te draaien.

## Setup
1. Open `index.html`.
2. Pas de `CONFIG` variabele aan met je eigen Telegram Bot Token en Chat ID.

## Hosting (Vercel)
Deze app is volledig statisch (HTML/JS/CSS in één bestand).
1. Push naar GitHub.
2. Importeer in Vercel.
3. Done.
